## §8. Red-flag Inventory

Items that, if true on a Monday review, escalate same-day:

- Any founder does NOT have a signed CIIAA.
- Any 83(b) election is past the 30-day window without filing.
- Any CIIAA or equity grant has been distributed but not counter-signed.
- Any external communication (press, pitch, paid ad) quotes a number that is not in `docs/v5_2_status/numeric_reconciliation.md`.
- Any security / trust / landing page claims a cert (SOC 2, ISO, GDPR) that has not been formally issued.
- Any creator onboarding in NYC without AEDT pre-use notice.
- Any USPTO publication of a mark similar to "ConversionOracle" or "Vertical AI for Local Commerce" by a third party.
