**Scan result** (`grep -oE "#[a-fA-F0-9]{6}" Design.md | sort -u`) found 20 hex values. The 14 beyond the six canonical colors include:

- `#000000`, `#010101` — pure black (used in documented "override-only" escape hatches, Design.md §Override rules)
- `#9ca3af`, `#6b7280`, `#4a5568` — Tailwind neutrals (cited in comparisons for "do NOT use")

**Conclusion**: no palette drift in production code. The "extras" in Design.md are explicit reference / comparison rows, not approved colors. Any new UI in this sprint MUST restrict to the six colors or a documented exception (e.g. `map-pin 50% radius`, `#ffffff` text on Flag Red).

**Go signal**: ✅ palette authority confirmed.
