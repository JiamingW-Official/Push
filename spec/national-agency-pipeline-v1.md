**Version**: v1 (v5.3 authoritative)
**Owner**: Jiaming
**Window**: Day 4–10 of v5.3 launch (2026-04-23 → 2026-04-29)
**Goal**: 8–12 intro outreach → 3–5 "interesting, tell me more" replies
**Intent**: relationship-building only — **no signing** expected at pre-seed

## 0. 为什么做这条线

National agency / platform 的决策周期是 **6–12 个月**，pre-seed 阶段签不下来。但如果到 Series A 才开始建关系，来不及 —— 这轮必须埋线。

本 pipeline 的定义的**成功**：
- 3–5 个 "interesting, tell me more" 层级的回复
- 1–2 个进入 15–30 分钟 Zoom 的认知建立
- 0 签约 —— 现在签约反而是警报（意味着我们要承诺 pilot 结果而我们没数据）

本 pipeline 的定义的**失败**：8–12 封 → 0 回复 → 重新设计 pitch 并回查 deliverability。

## 4. 回复处理 SOP

| 收到的回复 | 动作 |
|---|---|
| "Interested, tell me more" | 约 30 min Zoom，准备 3 张 slide（problem / Push / ask）；call 后 24h 内发跟进 + 1-pager |
| "Not now but interesting" | 加入 nurture list；Q3 2026 pilot 结果出来后再发 follow-up（附 case study） |
| "Competitive with us" | 思考是否真的 adjacent；回信承认立场 + ask if ever any way to collaborate；不再追 |
| "Send more info" | 发 1-pager（与 P1-1 agency playbook 同一份稍改）+ push-six-flax.vercel.app 链接；不放任何附件在首封 |
| "Wrong person, try [X]" | 立即转发给 [X]，CC 原人感谢转介 |
| No reply after 2 周 | 发一次 follow-up（subject 前加 "Re:"）；再 2 周仍无回复 → 归档 |

## 5. Outreach 发送时间

- **最佳时段**: Tue / Wed / Thu 上午 8–10 AM PST (11 AM – 1 PM EST) — 回复率实测最高
- **Avoid**: Monday AM (周一 inbox 爆炸), Friday PM (不看邮件), US 节假日前后 3 天
- **LinkedIn vs email**: LinkedIn 适合"接入点不明"的目标（HeyTea NA / Mixue）；email 适合已有通用邮箱的目标（Blue Bottle HQ, Ogilvy partnerships）

## 6. 失败容忍度

- 预期回复率: **10–15%** (pre-seed + unknown founder 的基线)
- 8–12 封 → 1–2 封有意义回复即成功
- 0 回复不触发 freeze，但 Day 10 review 时必须诊断：
  - Subject line 是否太长 / 太 spammy
  - Send time 是否落在 avoid 窗口
  - 发件域 (wangjiamingaas@gmail.com) 是否进了 spam
  - LinkedIn account 是否有足够 activity 证明非 bot

## 7. 不做的事（硬红线）

- 不发 mass 未个性化 email（会被标 spam，域名一次性破坏）
- 不在 outreach 里放附件（降低 deliverability；pdf / docx 一律链接替代）
- 不承诺 pilot 结果（没数据就不做 claim — 措辞用 "piloting" 不用 "proven"）
- 不用 template 后忘改人名 / 公司名 / "[Brand]" 占位符（列入每次发送前 checklist）
- 不追 national 目标超过 2 次 follow-up（尊重他们的沉默 = 尊重自己的时间）
- 不在同一封邮件里发给 multiple recipients（即使想 CC 同公司两个人）
- 不在 Meta / Google / Snap 的公司邮箱 cold 联系运营岗位之外的人（易触发内部垃圾规则）

## 9. Day 10 Review 触发的决策

- **Healthy**: ≥ 3 positive replies → 进入 Q2 nurture 阶段，每月 1 次 touch
- **Warning**: 1–2 positive replies → 继续 nurture 但不扩规模
- **Red**: 0 positive replies → 重新诊断 pitch，暂停 national，全力 NYC local 直签
- **Bonus**: 任何 "let's talk M&A / strategic" 级别回复 → 立即升级处理，正式走 advisor / counsel 流程
