## 0. Why agencies — the differentiated value

Push 对 agency 的差异化价值**不是**"便宜的 creator 名单"（agency 自己就有名单）。

Push 的差异化是 **"验证过转化率的 top creator + neighborhood-level 归因报告"** —— 这是 agency 现在盲配 creator 给 F&B 客户时的真实痛点：他们拿不到"这个 creator 的一个 post 带来了多少客人进店"的数据。

Push = agency 现有流程的 **reporting layer**，不抢 agency 的位置。

## 2. Push 的 Value Prop（对 agency）

**One-liner**: "We give your F&B accounts what TikTok and Instagram never could — verified offline conversion data tied to creator campaigns."

### The 3 asks we can deliver

1. **Creator performance reports** — 每个 creator 在 Push 的实到转化率、ROI、retention；agency 可以据此决定 matchmaking
2. **Neighborhood intelligence** — 每月给一份 SoHo / Williamsburg / LES 的 F&B 消费流向 report，agency 可以用来给客户写 pitch
3. **Top-tier creator access** — Push 的 T4–T6 验证 creator pool 对 agency open，优先级别客户

### What we ask in return

- Agency 把 Push 推给他们现有 F&B 商家客户，当作 "conversion attribution layer"
- Agency 分成：Push 从 licensing fee 里给 agency **15–20% rev share**
- Agency 成为 Push 某一个 neighborhood 的 "preferred partner"（非独家，但优先）

## 6. Discovery Call Script (15 分钟)

| Time | Segment | 动作 |
|---|---|---|
| 0–2 min | Intro | Hi, brief who we are, why we're calling |
| 2–5 min | Ask | "How do your F&B clients currently measure creator campaign success?" — listen for vanity metrics (reach / views) vs. actual revenue attribution. **This is the pain we solve.** |
| 5–10 min | Show | 1 slide pilot dashboard mockup (creator ROI by store) |
| 10–13 min | Propose | 90-day pilot LOI |
| 13–15 min | Next step | Schedule follow-up with their senior lead |

### What to listen for (signal taxonomy)

- **GREEN**: "Yeah, we can't show clients real ROI, only engagement" → proceed to LOI fast
- **YELLOW**: "We already use [tool X]" → position Push as complementary reporting layer, not replacement
- **RED**: "We just need cheaper creators" → politely disqualify, Push is not a price tool

## 8. Failure modes

| If | Then |
|---|---|
| 5 家 0 回复 (Day 5) | Freeze trigger — 重新设计 pitch，检查 subject line、send time |
| 收到回复但觉得 Push 太早期 | 提议"数据换数据"——你给我们看 1 个匿名客户 campaign，我们免费跑 pilot |
| Agency 要求独家 neighborhood | 可以给，但要求 minimum volume commitment（3 merchants / 季） |
| Agency 想要 raw data export | 拒绝，只给 aggregated report + API access |

## 10. Success / Freeze criteria (Day 10 end-of-window)

- **SUCCESS**: ≥ 1 signed LOI → v5.3 双轨 GTM 第 1 轨启动，转入 90-day 共同 pilot
- **PARTIAL**: 0 LOI but ≥ 2 discovery calls with YELLOW signal → extend 2 周并迭代 pitch
- **FREEZE**: 0 discovery calls by Day 10 → 停止 agency 轨，全力直签 merchants；复盘 pitch / target list
