### C-1. 添加 Champagne + Pearl Stone 使用警告

**文件：** `.claude/skills/push-website/SKILL.md` → Color Palette 章节末尾

添加：
```markdown
### Color Combination Warning
- **Pearl Stone `#f5f2ec` + Champagne `#c9a96e` 同时大面积出现时整体偏暖黄。** 在 Champagne 高亮元素旁边务必有一个冷色锚点（`--dark` 或 `--tertiary`）做平衡。

```

```markdown
- **Champagne + Surface 搭配警告：** 当 `--champagne` 元素出现在 `--surface` 背景上时，确保附近有冷色（`--dark`/`--tertiary`/`--graphite`）元素作为视觉锚点，防止整体偏暖。Champagne 一个页面内不超过 3 处使用。
```
